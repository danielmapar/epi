# Elements of Programming Interviews in C++ - Notes

# Primitive Types

operator	asm equivalent	description
* `&`	AND	Bitwise AND
* `|`	OR	Bitwise inclusive OR
* `^`	XOR	Bitwise exclusive OR
* `~`	NOT	Unary complement (bit inversion)
* `<<`	SHL	Shift bits left
* `>>`	SHR	Shift bits right
